#College Administration System (CE Poonjar)

Developed a comprehensive College Administration System for my institution using the MERN stack (MongoDB, Express.js, React, Node.js). This system streamlines administrative, academic, and student-related tasks through separate modules for faculty, students, and administrators, ensuring seamless operation across all roles.

Faculty Module: Allows faculty members to efficiently manage their responsibilities by providing features to take attendance, input student marks, and create question papers. The interface is user-friendly and designed for ease of use, helping faculty save time on routine tasks.

Admin Module: Designed for the institution's principal, this module offers complete control over academic operations. Admins can monitor and manage all classes, view attendance reports, and oversee the academic performance of students across different departments.

Student Module: Enables students to access their academic details in real time. They can view their marks for individual subjects, track their attendance for each class, and even check attendance for specific dates. This promotes transparency and helps students stay informed about their progress.

The system offers a high level of customization and responsiveness, providing a smooth experience across different devices. By leveraging the power of the MERN stack, the project ensures scalable, fast, and secure performance for institutional needs.
